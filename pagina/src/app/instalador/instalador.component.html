Instalador

Este tutorial explica como funciona o processo de instalação da integração Protheus & Auditto. Esse programa criará os dicionários de dados de estrutura para processamento da integração.
Abra o smartclient e no lugar da chamada de programa inicial preencha U_ADTINS001 e clique em Ok.
Selecione o produto, entre: Todos, XML - Para entrada de documentos, APR - Apuração para o cruzamento de apurações. Informe o usuário e senha da API que foram enviados pela auditto, caso deseje não instalar em alguma empresa, é só alterar a seleção.
Caso selecione o XML ou Todos, aparecerá uma outra tela de parâmetros para informar as tabelas que serão utilizadas na tua empresa, ele vai criar três tabelas, uma para o cabeçalho, outra para os itens e uma para as duplicatas:
Aguarde o término do processamento.
Recomendação: Confira a geração dos dicionários de dados SX2, SX3, SIX e SX6.
