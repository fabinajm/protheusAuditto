Protheus

    Caso use Pedido de compras, existem dois parâmetros que podem ser configurados:
  • ADT_PCOPER - Campo de operacao do Pedido de Compras
  • ADT_NUMPRC - Campo de numero de processo do Pedido de Compras
    Código de produto para entrada de documentos, será utilizado apenas quando não se usa pedido de compras e nem documento original:
  • ADT_NFSPRD - Código de produto para entrada de Nota Fiscal de Serviço
  • ADT_CTEPRD - Código de produto para entrada de Documento de Transporte
  • ADT_NDPRD - Código de produto para entrada de Nota de Débito
    Caso use TES Inteligente ou deseje por algum motivo não utilizar as regras fiscais do Auditto, existem dois parâmetros:
  • ADT_TESNEW - Cria a TES senão encontrar conforme regras do Auditto ?
  • ADT_TESALT - Altera TES caso encontre mas tenho algo diferente em relação as regras do Auditto ?
    O sistema gera log no console.log de todo o passo de integração e é possível também que gere um json de todo o arquivo que é enviado, para isso existem dois parâmetros:
  • ADT_GERLOG - Gera LOG de processamento no console.log ?
  • ADT_GERJSN - Gera o json dos cadastros e movimentos no protheus_data para log ?
